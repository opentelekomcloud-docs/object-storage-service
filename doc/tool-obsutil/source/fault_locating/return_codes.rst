Return Codes
============

If obsutil is invoked by processes, the command output cannot be viewed in real time. obsutil generates different return codes based on different execution results. :ref:`Table 1 <obs_11_0058__table936012247195>` describes the return codes. You can use either the following methods to obtain the return code of the latest execution result and then analyze and rectify the fault based on it:

-  In the macOS or Linux OS, run the following command to obtain the return code of the latest execution result:

   .. code-block::

      echo $?

-  In the Windows OS, run the following command to obtain the return code of the latest execution result:

   .. code-block::

      echo %errorlevel%
