Moving a File or Folder
=======================

Move a file or folder stored in OBS from one path to another path.

Procedure
---------

#. Log in to OBS Browser+.
#. Go to the target bucket. Right-click the file or folder and choose **Move** from the shortcut menu.
#. Right-click the path for saving the file or folder and choose **Paste** from the shortcut menu.

   .. note::

      -  Files or folders in external buckets cannot be moved.
      -  The source path and target path must be different.
